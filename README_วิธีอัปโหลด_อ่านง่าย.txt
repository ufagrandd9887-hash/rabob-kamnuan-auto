แพตช์นี้ปรับข้อความหน้าเลขแนะนำให้คนดูหวยเข้าใจง่ายขึ้น

ไฟล์ที่ให้อัปโหลดทับใน GitHub:
- index.html
- netlify.toml ถ้ามีในโฟลเดอร์นี้

วิธีทำ:
1) แตกไฟล์ ZIP
2) เข้า GitHub repo เดิม
3) Add file > Upload files
4) ลาก index.html และ netlify.toml ไปวางทับของเดิม
5) Commit changes
6) Netlify > Deploys > Trigger deploy > Deploy site
7) เปิดเว็บแล้วกด Ctrl+F5

ผลที่ควรเห็นในหน้าแรก:
- หัวข้อ เลขน่าตามจากผลงวดเก่า
- ระบบดูจากผลงวดเก่ากี่งวด
- ตั้งแต่วันที่ไหนถึงวันที่ไหน
- แต่ละเลขบอกว่าเคยออกจริงล่าสุดวันไหน
- ผลวันนั้นออกอะไร
- เคยออกกี่ครั้งในงวดเก่าที่เปิดดู
